Nginx 输出在线日志

Posted by Sunday on 2018-05-31

nginx输出node pm2日志，实现浏览器查看日志内容(默认是下载)

root@demo18:/data/logs/pm2# ls -l

root@demo18:/data/logs/pm2# cat /etc/nginx/other/log_output.types

root@demo18:/data/logs/pm2# cat /etc/nginx/vhost/logs.sundayle.com.conf
